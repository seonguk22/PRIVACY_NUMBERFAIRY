# 숫자 요정 개인정보처리방침 (Korean)

**시행일:** 2025년 12월 09일

**숫자 요정** (이하 "서비스")는 사용자의 개인정보를 소중하게 생각하며, 관련 법령을 준수합니다. 본 개인정보처리방침은 사용자가 앱을 이용할 때 수집되는 정보와 이용 방법을 설명합니다.

## 1. 수집하는 개인정보 항목

본 앱은 별도의 회원가입 절차(이메일, 전화번호 수집 등) 없이 **익명 로그인(Anonymous Login)** 방식을 사용합니다.

### A. 자동으로 수집되는 정보
* **Firebase Authentication (익명):** 사용자를 구분하고 데이터를 동기화하기 위한 고유 식별자(User ID)
* **생성 및 기록 데이터:** 사용자가 생성한 로또 번호 조합, 즐겨찾기(찜) 목록, 생성 기록 등
* **기기 정보:** 기기 모델명, OS 버전, 언어 설정, 타임존
* **광고 식별자 (Advertising ID):** Google AdMob 광고 제공을 위해 수집
* **FCM 토큰 (Firebase Cloud Messaging):** 로또 당첨 결과 알림 발송을 위한 기기 식별 토큰

### B. 사용 권한 (Android)
* **알림 (POST_NOTIFICATIONS):** 매주 토요일 당첨 결과 및 서비스 관련 알림 전송
* **인터넷 (INTERNET):** 로또 당첨 번호 데이터 조회 및 서버 통신

## 2. 개인정보의 이용 목적

* **서비스 제공:** 로또 번호 생성, 즐겨찾기 저장 및 관리, 통계 분석 데이터 제공
* **사용자 식별:** 익명 ID를 통한 기기 간 데이터 동기화 및 유지
* **알림 전송:** 매주 토요일 당첨 결과 푸시 알림 제공
* **광고 게재:** Google AdMob을 통한 배너, 전면, 보상형 광고 제공
* **서비스 개선:** 오류 분석 및 앱 성능 최적화

## 3. 제3자 서비스 제공업체

본 앱은 다음의 제3자 서비스(SDK)를 사용하며, 해당 업체들은 각자의 정책에 따라 데이터를 처리합니다.

* **Google Firebase:** 데이터 저장(Firestore), 인증(Auth), 푸시 알림(Cloud Messaging), 분석(Analytics) ([개인정보처리방침](https://firebase.google.com/support/privacy))
* **Google AdMob:** 맞춤형 광고 게재 ([개인정보처리방침](https://policies.google.com/technologies/ads))

## 4. 데이터의 보유 및 삭제

### A. 보유 기간
데이터는 사용자가 서비스를 이용하는 동안 Firebase 서버에 안전하게 보관됩니다.

### B. 데이터 삭제 (회원 탈퇴)
앱 내 **[설정] > [데이터 삭제 및 탈퇴]** 버튼을 통해 즉시 모든 데이터를 영구 삭제하고 서비스를 탈퇴할 수 있습니다. 삭제된 데이터는 복구할 수 없습니다.

### C. 자동 파기
사용자의 개인정보 보호 및 서버 리소스 관리를 위해, **90일 이상 앱에 접속하지 않은 장기 미사용 계정**의 모든 데이터(즐겨찾기, 생성 기록 등)는 자동으로 영구 삭제됩니다.

## 5. 아동의 개인정보 보호

본 서비스는 만 13세 미만 아동의 개인정보를 고의로 수집하지 않습니다. 만약 수집 사실이 확인될 경우 즉시 해당 정보를 삭제 조치합니다.
